## 📌 Objetivo do Projeto
O objetivo deste desafio foi desenvolver um modelo capaz de prever a nota IMDB de filmes com base em características como gênero, faturamento, elenco e diretor. O projeto simula uma demanda real de análise de dados, focando em precisão estatística e geração de insights acionáveis.

## 🛠️ Stack Tecnológica
* **Linguagem:** Python
* **Análise de Dados:** Pandas, NumPy
* **Visualização:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn
* **Persistência de Modelo:** Pickle / Joblib

## ⚙️ Metodologia Aplicada
1. **Análise Exploratória (EDA):** Estudo detalhado sobre o impacto das variáveis (como faturamento e elenco) na recepção do público.
2. **Tratamento de Dados:** Saneamento de valores ausentes, codificação de variáveis categóricas (Encoding) e normalização.
3. **Modelagem:** Implementação e comparação de algoritmos de Regressão para otimização das métricas de erro (MAE/RMSE).
4. **Relatório Técnico:** Elaboração de documento executivo detalhando as descobertas e a performance do modelo final.

## 📈 Resultados e Entregáveis
* **Modelo Serializado:** O arquivo `modelo_imdb.pkl` contém o modelo treinado e pronto para realizar inferências em novos dados.
* **Documentação Executiva:** O arquivo `Relatório.pdf` detalha a jornada analítica, justificando as escolhas técnicas e apresentando os principais insights extraídos.

## 🏭 Conexão com a IA Industrial
A técnica de **Regressão Multivariada** aplicada neste projeto é a mesma utilizada na indústria para **Previsão de Qualidade** e **Otimização de Rendimento**. Prever um indicador (nota) baseado em múltiplos fatores de entrada é o fundamento para prever a qualidade final de um produto industrial com base em variáveis de processo.
